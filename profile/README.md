## Welcome to Knowledge Engineering @ UoL 👋

We are part of the [Neuro-symbolic AI and knowledge engineering](https://www.liverpool.ac.uk/computer-science-and-informatics/research/artificial-intelligence/neuro-symolic-ai-knowlege-engineering/) research group at the University of Liverpool. You can find references to our software, libraries, and datasets below, along with their documentation and instructions for use. If you have any questions, please feel free to get in touch with us.

### 💻 Software and libraries

| **Name** | **Description** | **Resources** |
|---|---|---|
| &nbsp; | &nbsp; | &nbsp; |
| *RELRaE v2* | A second iteration of the RELRaE framework for creating a skeleton ontology from an XML schema. | [Repo](https://github.com/KE-UniLiv/RELRaE_2)<br />[Article](#) |
| *RELRaE* | An LLM-Based Relationship Extraction, Labelling, Refinement, and Evaluation framework to generate a skeleton ontology from an XML schema. | [Repo](https://github.com/KE-UniLiv/RELRaE)<br />[Article](#) |
| *RuBREx* | Rules-Based Relationship Extraction module for XML schema concepts. | [Repo](https://github.com/KE-UniLiv/RuBREx)<br />[Article](#) |
| *IDEA2* | Expert-in-the-loop requirement elicitation and analysis for ontology engineering. | [Repo](https://github.com/KE-UniLiv/IDEA2)<br />[Article](#) |
| *CQStoryRAG* | RAG-based digital storytelling from Music Heritage Knowledge Graphs. | [Repo](https://github.com/KE-UniLiv/cq-storyrag)<br />[Article](#) |
| *CompCQ* | A computational framework for empirical analysis of Competency Questions. | [Repo](https://github.com/KE-UniLiv/compcq)<br />[Article](#) |

### 💾 Datasets, Ontologies & KGs

| **Name** | **Description** | **Resources** |
|---|---|---|
| &nbsp; | &nbsp; | &nbsp; |
